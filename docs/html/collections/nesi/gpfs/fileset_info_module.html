nesi.gpfs.fileset_info module – Provides information about Spectrum Scale fileset

Note

This module is part of the nesi.gpfs collection (version 1.0.0).

To install it, use: ansible-galaxy collection install nesi.gpfs.

To use it in a playbook, specify: nesi.gpfs.fileset_info.

  • Synopsis

  • Parameters

  • Examples

  • Return Values

Synopsis

  • The same output as mmlsfileset -Y provided as a dictionary

Parameters

Parameter

Comments

filesystem

string / required

Spectrum Scale filesystem name.

name

string / required

name of the fileset.

Examples

- name: Info for projects fileset
  nesi.gpfs.fileset_info:
   filesystem: filesystem1
   name: projects
  register: fs_result
